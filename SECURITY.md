# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes    |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability, please report it through [GitHub's private security advisory feature](https://github.com/bitfootco/skogen/security/advisories/new).

You can expect:
- An acknowledgement within 48 hours
- A status update within 7 days
- A CVE and patch release for confirmed vulnerabilities as quickly as practicable

We ask that you give us a reasonable amount of time to address the issue before public disclosure.
